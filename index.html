Projects I have done

HandWritten Character Recognizer

This project aims at developing a system which will recognize characters of English language from a given scanned document. The system takes a document with handwritten English characters as input, processes the characters, trains the classifier and simulate the classifier to recognize the characters. The accuracy of system is 86%.

Exploropedia

Exploropedia a web application which assists a person who is planning to visit or know about a city. It displays complete list of all the hotels in that city with information like ratings and reviews. The application also lists top tourist attractions and ‘things to do’ in the city with detailed description. It would give better results than many of the existing E-tourism websites for small or not-so-popular cities e.g. Amethi.

Tourist Navigation System

It is a web based application intended for tourists. A tourist, who is visiting a new city, can have many requirements such as sight-seeing, shopping, having some specific food etc. The application suggests the user shortest itinerary fulfilling all these requirements. Thus the tourist saves her time and effort by travelling on the shortest path.
